# ATACUT Resources

This directory contains custom fonts and video transitions for the ATACUT video editor.

## Transitions

Located in `resources/transitions/`, includes 12 professional video transitions from Mixkit:

### Available Transitions
1. **Zoom** - Simple zoom effect
2. **Zoom Bounce** - Bouncy zoom transition
3. **Outwards Zoom** - Zoom from center outwards
4. **Sideways Zoom** - Diagonal zoom effect
5. **Countdown Zoom** - Numbered countdown zoom
6. **Spin & Twist** - Rotating twist effect
7. **Reverse Spin** - Counter-rotating effect
8. **Distort Sideways** - Warped side transition
9. **Liquid Swipe** - Fluid swipe left to right
10. **Bright Bokeh** - Bokeh light transition
11. **Twirling Bokeh** - Spinning bokeh lights
12. **Digital Static** - Glitch/static effect

### Classic Transitions
- Fade
- Dissolve
- Wipe
- Slide
- Crossfade

### Usage
Right-click on a clip in the timeline and select "Add Transition" to choose from available effects. Transitions are applied between clips.
